git config --global user.email raffel222@gmail.com - konfiguracja użytkownika przez email

git init - init a local repository with .git folder

git remote add origin "<link>" - bind workspace with remote repository

git pull origin <branch> - pobierz pliki z repozytorium

clear - wyczyść terminal

git status - wyświetl obecny status plików
On branch master
Untracked files:
  (use "git add <file>..." to include in what will be committed)
        git instructions.txt

nothing added to commit but untracked files present (use "git add" to track)

git add <file> - dodaj plik do indeksu (pośrednik między workspace a lokalnym repozytorium)
git add -A     - dodaj wszystkie pliki z brancha do indeksu

git commit -m "<wiadomość>" - wprowadź zmiany do lokalnego repozytorium
gti commit -a -m "adding three files at once" - wprowadzenie kilku plików/zmian do lokalnego repozytorium

git log - log wprowadzonych operacji

git branch <branchName> - utworzenie nowego brancha

git checkout <branchName> - zmiana brancha na <branchName>
